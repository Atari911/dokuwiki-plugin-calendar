=== EventList Range Parameter ===

The {{eventlist}} widget now supports a simple "range" parameter!

== Syntax ==

{{eventlist range=day}}
{{eventlist range=week}}
{{eventlist range=month}}
{{eventlist range=day namespace=personal}}
{{eventlist range=week namespace=work}}

== Options ==

**range=day**
  * Shows: Today's events only
  * Header: "Today"
  * Example: {{eventlist range=day}}

**range=week**
  * Shows: Next 7 days (today + 6 days)
  * Header: "This Week"
  * Example: {{eventlist range=week}}

**range=month**
  * Shows: Current month (1st to last day)
  * Header: "February 2026" (current month name)
  * Example: {{eventlist range=month}}

== Combining with Namespace ==

{{eventlist range=day namespace=work}}
  → Today's work events

{{eventlist range=week namespace=personal}}
  → This week's personal events

{{eventlist range=month namespace=*}}
  → This month's events from all namespaces

== Legacy Options (Still Work) ==

**today** - Same as range=day
{{eventlist today}}

**sidebar** - Shows today through +1 month
{{eventlist sidebar}}

**date=YYYY-MM-DD** - Specific date
{{eventlist date=2026-02-15}}

**daterange=YYYY-MM-DD:YYYY-MM-DD** - Custom range
{{eventlist daterange=2026-02-01:2026-02-28}}

== Examples ==

**Dashboard sidebar:**
{{eventlist range=day namespace=*}}
{{eventlist range=week namespace=work}}

**Project page:**
{{eventlist range=month namespace=projects:website}}

**Quick view:**
{{eventlist range=day}}  ← Today
{{eventlist range=week}} ← This week
{{eventlist range=month}} ← This month
