====== Calendar Plugin Examples - DokuWiki Syntax ======

Copy and paste these examples directly into your DokuWiki pages.

===== Basic Usage =====

==== Simple Current Month Calendar ====

<code>
====== Office Calendar ======

{{calendar}}
</code>

This displays a compact 800x600 calendar for the current month with an integrated event list on the right.

==== Specific Month and Year ====

<code>
====== June 2026 Planning ======

{{calendar year=2026 month=6}}
</code>

==== Multiple Months on One Page ====

<code>
====== Q1 2026 Calendar ======

===== January =====
{{calendar year=2026 month=1}}

===== February =====
{{calendar year=2026 month=2}}

===== March =====
{{calendar year=2026 month=3}}
</code>

==== Event Panel Only (Right Panel) ====

<code>
====== Team Events Panel ======

{{eventpanel namespace=team}}
</code>

Display just the 320px wide event management panel without the calendar grid. Perfect for sidebars!

==== Event Panel with Navigation ====

<code>
====== Marketing Events ======

{{eventpanel year=2026 month=6 namespace=marketing}}

**Features**:
  * Navigate months with ‚óÑ ‚ñ∫ buttons
